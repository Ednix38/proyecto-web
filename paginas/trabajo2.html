Configuración de Entorno de Desarrollo Web

Fecha de entrega: 12 de septiembre de 2025 • Tipo: Investigación escrita • Unidad I

Descripción

El trabajo consiste en describir paso a paso cómo instalar y configurar un entorno de desarrollo web usando Visual Studio Code (VS Code), orientado al desarrollo con HTML, CSS y JavaScript.

Objetivo

Permitir que cualquier persona, incluso sin experiencia previa, configure un entorno funcional y eficiente para empezar a programar sitios web.

Introducción

El desarrollo web moderno requiere de herramientas que faciliten escribir, organizar y ejecutar código de manera rápida y eficiente. Visual Studio Code es un editor ligero, gratuito y multiplataforma creado por Microsoft, que se ha convertido en el favorito de principiantes y profesionales. Este trabajo detalla la instalación, configuración inicial y creación de un primer proyecto web con VS Code.

1. Descarga e instalación de VS Code

1.1 Acceso al sitio oficial

Descarga VS Code desde su página oficial: https://code.visualstudio.com/. El sitio detecta automáticamente tu sistema operativo (Windows, macOS o Linux).

1.2 Instalación según sistema operativo

  • Windows:
    1. Descargar el archivo .exe.
    2. Ejecutar el instalador y aceptar los términos.
    3. Seleccionar carpeta de instalación (recomendado predeterminada).
    4. Opcional: agregar VS Code al menú contextual y PATH.
    5. Hacer clic en "Instalar".
  • macOS:
    1. Descargar archivo .dmg.
    2. Arrastrar ícono de VS Code a la carpeta Aplicaciones.
    3. Abrir desde Launchpad o Spotlight.
  • Linux (Ubuntu/Debian):
    1. Descargar paquete .deb.
    2. Instalar con doble clic o comando: sudo dpkg -i code_versión.deb
    3. Distribuciones Red Hat/Fedora usan paquete .rpm.

2. Explorando la interfaz de VS Code

Al abrir VS Code, la interfaz principal incluye:

  • Barra lateral izquierda: Archivos, búsqueda, control de extensiones.
  • Área principal: Editor de archivos.
  • Barra inferior: Información de codificación, errores, puerto de Live Server.
  • Marketplace de extensiones: Instalar plugins y funcionalidades adicionales.

3. Personalización inicial del editor

3.1 Cambiar tema de color

  1. Presionar Ctrl + Shift + P (Cmd + Shift + P en Mac).
  2. Escribir "Color Theme".
  3. Seleccionar tema predeterminado o descargar del Marketplace.

Ejemplos de temas populares:

  • Dark+ (oscuro, predeterminado)
  • Light+ (claro)
  • One Dark Pro
  • Dracula Official
  • Monokai Pro

3.2 Iconos de archivos

  • Instalar extensión Material Icon Theme.
  • Activar desde la configuración como tema de iconos.

4. Extensiones esenciales para desarrollo web

  • Live Server: abre archivos HTML y actualiza automáticamente al guardar.
  • HTML Snippets: fragmentos de código HTML predefinidos.
  • JavaScript (ES6) Code Snippets: atajos para funciones modernas de JS.
  • CSS Snippets: acelera la escritura de reglas CSS.
  • Prettier – Code Formatter: formatea el código automáticamente al guardar.

5. Creación del primer proyecto web

5.1 Crear carpeta y archivos

  1. Crear carpeta llamada "Aplicaciones Web".
  2. Abrirla con VS Code.
  3. Crear archivo index.html.

5.2 Código HTML inicial

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primer sitio</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
  <p>Este es mi primer proyecto con VS Code.</p>
</body>
</html>

5.3 Probar con Live Server

  • Hacer clic derecho en index.html.
  • Seleccionar "Open with Live Server".
  • El navegador mostrará la página y se actualizará automáticamente al guardar cambios.

6. Configuración básica para mayor comodidad

  • Autoguardado: Archivo → Preferencias → Configuración → "Auto Save" → Activar.
  • Formateo automático: Instalar Prettier y activar "Format on Save".

7. Recomendaciones finales

  • Organizar el proyecto en carpetas: /css, /js, /img.
  • Probar cambios constantemente con Live Server.
  • Instalar nuevas extensiones conforme se avance, evitando sobrecargar el editor al inicio.

Conclusión

La instalación y configuración de VS Code permite contar con un entorno profesional y amigable en minutos. Con las extensiones adecuadas y prácticas como Live Server y Prettier, incluso un principiante puede crear su primer sitio web funcional y comenzar a aprender desarrollo web de manera efectiva. Con el tiempo, se pueden integrar herramientas adicionales como control de versiones con Git y frameworks modernos.
