Conceptos Generales del Desarrollo Web

Fecha de entrega: 5 de septiembre de 2025 • Tipo: Investigación escrita • Unidad I

Descripción

Identificar los conceptos fundamentales de las aplicaciones web: Internet y Web, aplicación web, front end, back end, hosting, protocolos (HTTP, HTTPS) y el modelo cliente-servidor.

Objetivo

Comprender los componentes esenciales que forman el ecosistema del desarrollo web y cómo interactúan para crear aplicaciones funcionales y accesibles en la red.

Contenido

  • Internet y Web: Infraestructura de comunicación que conecta millones de dispositivos.
  • Aplicación Web: Software accesible desde un navegador mediante un servidor remoto.
  • Front-End: Parte visible de la aplicación (HTML, CSS, JavaScript).
  • Back-End: Lógica del servidor, base de datos y API.
  • Hosting: Servicio que almacena y publica los sitios web en Internet.
  • Protocolos HTTP/HTTPS: Normas de comunicación entre cliente y servidor.
  • Modelo Cliente-Servidor: Arquitectura donde un cliente solicita y un servidor responde.

Ampliación de Contenido

Introducción

La web se ha integrado profundamente en nuestra vida, transformando la forma en la que nos comunicamos, trabajamos, aprendemos y accedemos a servicios esenciales. Prácticamente cualquier actividad, como gestiones bancarias, consultas médicas, educación a distancia, entretenimiento, entre muchas otras dependen directa o indirectamente de aplicaciones y servicios alojados en internet. La tecnología nos hace entender que sus fundamentos son cada vez más relevantes, no solo para especialistas sino para cualquier persona que interactúe con dispositivos digitales.

Detrás de cada página web, plataforma o servicio en línea existe una arquitectura tecnológica compleja. Conceptos como el funcionamiento de internet, el rol de los protocolos de comunicación, la estructura cliente-servidor y los lenguajes de programación web conforman los pilares esenciales que sostienen la experiencia digital actual. Comprender estos elementos nos permite apreciar la ingeniería detrás de interfaces aparentemente simples, y al mismo tiempo, desarrollar una visión más crítica y consciente sobre el ecosistema digital que moldea nuestra realidad.

Internet: La Red de Redes

El Internet es una infraestructura global de redes de computadoras interconectadas que utilizan un conjunto común de protocolos (TCP/IP) para comunicarse. No es lo mismo que la Web; Internet es la carretera, mientras que la Web es uno de los muchos servicios (como el correo electrónico o la mensajería instantánea) que circulan por ella. Es una red descentralizada, lo que significa que no tiene un núcleo central controlador, haciéndola robusta y resistente a fallos.

Servicios habilitados:

  • Email (SMTP/POP/IMAP)
  • Transferencia de Archivos (FTP/SFTP)
  • Mensajería Instantánea y VoIP
  • Acceso Remoto (SSH/Telnet)

Funcionamiento del Internet

  • Reglas de Empaquetamiento (Protocolos): Los datos se fragmentan en paquetes; TCP asegura entrega y orden.
  • Medio Físico (Red): Cables, routers, torres móviles y satélites. Cada dispositivo tiene una IP única.
  • Enrutamiento (Routing): Routers deciden la mejor ruta para cada paquete usando IP.

Familia de Protocolos TCP/IP

Conjunto de reglas que permite la comunicación entre dispositivos en Internet. Incluye:

  • TCP: entrega confiable y ordenada
  • IP: direccionamiento y enrutamiento
  • HTTP: protocolo de páginas web
  • DNS: traduce nombres de dominio a IP

Otros Protocolos de la Web

  • ARP: Traduce IP a MAC
  • DHCP: Asigna IP automáticamente
  • TLS/SSL: Cifrado seguro (HTTPS)

WWW (World Wide Web)

Sistema de información basado en hipertexto. Funciona con modelo cliente-servidor, usando HTTP/HTTPS.

Componentes Fundamentales

  • URLs
  • HTTP
  • HTML
  • Hipertexto

Modelo Cliente-Servidor

  • Cliente: navegador o app que solicita recursos.
  • Servidor: computadora que entrega los recursos solicitados.

Métodos HTTP

  • GET, POST, PUT, DELETE, HEAD, OPTIONS

Aplicaciones Web

Programas accesibles mediante navegador, desde páginas estáticas hasta SPAs complejas.

Modelo de 3 capas:
  • Front-end: HTML, CSS, JS, UI
  • Back-end: lógica del negocio y servidor
  • Capa de datos: bases de datos relacionales o NoSQL

Tipos de Aplicaciones Web

  • Basadas en navegador
  • Basadas en cliente
  • Aplicaciones móviles
  • Servicios web (APIs)

Ventajas y Retos

Accesibilidad, actualización centralizada, multiplataforma, seguridad, compatibilidad, SEO, escalabilidad.

AJAX

Permite actualizar partes de la página sin recargarla. Base de SPAs.

Web Stacks Comunes

  • LAMP
  • MEAN/MERN
  • JAMstack

Herramientas del Desarrollador

Editores: VSCode, WebStorm, Sublime Text. Control de versiones: Git. Frameworks: React, Angular, Vue.js, Laravel, Django, Express, Rails, Spring.

Principios de Desarrollo

  • KISS
  • DRY
  • YAGNI

Codificación y Formatos

  • UTF-8
  • JSON
  • XML

W3C y Estándares Web

Importancia de HTML, CSS, accesibilidad, SEO, interoperabilidad y mantenibilidad.

Conclusión

El internet y la web son ecosistemas fundamentales para la comunicación, la educación y el desarrollo tecnológico. Comprender su arquitectura y fundamentos permite crear aplicaciones eficientes, seguras y escalables, así como participar críticamente en el futuro digital.
