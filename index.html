Aplicaciones Web — Resumen de la materia

En esta materia se estudian las tecnologías básicas para construir páginas y aplicaciones para la web: HTML, CSS y JavaScript. A continuación se presenta un resumen de los conceptos, competencias y objetivos de la materia.

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que estructura el contenido de una página: encabezados, párrafos, listas, tablas, imágenes y enlaces. HTML define la semántica del contenido para navegadores y para herramientas de accesibilidad.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de estilos que controla la apariencia: colores, tipografías, distribuciones (layout), y comportamiento responsivo. Con CSS hacemos que la página se vea bien en distintos tamaños de pantalla.

¿Qué es JavaScript?

JavaScript es el lenguaje que añade interactividad y comportamiento: responder a eventos, modificar el DOM, consumir APIs, validar formularios y crear experiencias dinámicas en el navegador.

Competencias de la materia

  • Diseñar y estructurar páginas web utilizando HTML de manera semántica.
  • Aplicar estilos y layouts responsivos con CSS.
  • Implementar interactividad y validaciones con JavaScript.
  • Integrar buenas prácticas de accesibilidad y usabilidad.
  • Desarrollar proyectos web completos, combinando teoría y práctica.

Objetivos de aprendizaje

  1. Comprender la estructura básica de un documento HTML.
  2. Aplicar CSS para mejorar la presentación visual de contenidos web.
  3. Manipular el DOM y responder a eventos usando JavaScript.
  4. Desarrollar habilidades de depuración y prueba de aplicaciones web.
  5. Fomentar la creatividad y la resolución de problemas en proyectos web.

Actividades y proyectos

Durante el curso se realizan diversas actividades prácticas, entre las que destacan:

  • Creación de páginas web estáticas con HTML y CSS.
  • Integración de scripts de JavaScript para formularios y validaciones.
  • Desarrollo de mini proyectos individuales y grupales.
  • Presentación de un portafolio digital con todos los ejercicios realizados.

Habilidades adquiridas

Al finalizar la materia, el estudiante podrá:

  • Construir páginas web accesibles y responsivas.
  • Implementar interactividad básica mediante JavaScript.
  • Aplicar estándares y buenas prácticas de desarrollo web.
  • Documentar y presentar proyectos de manera profesional.

Propósito de la materia

El propósito de este curso es que el estudiante sea capaz de concebir, construir y publicar páginas web accesibles y responsivas, aplicando buenas prácticas de semántica, diseño y código. Se trabajan tanto ejercicios teóricos como prácticas y proyectos, fomentando la creatividad, la organización y el pensamiento lógico.
